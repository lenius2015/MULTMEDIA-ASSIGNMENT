# E-Commerce Homepage Design Documentation

## Overview
This document describes the modern, professional e-commerce homepage design inspired by platforms like Kikuu and Alibaba. The design features a clean, responsive layout with smooth animations and user-friendly interactions.

## Design Features

- **Location**: Top of the page
- **Features**:

  - Eye-catching gradient background
  - Animated shipping icon
  - Fully responsive

#### Main Navigation
- **Logo**: 
  - Shopping bag icon with brand name "OMUNJU SHOPPERS"
  - Hover effect with scale animation
  
- **Search Bar**:
  - Category dropdown selector
  - Large search input field
  - Search button with icon
  - Focus state with border highlight

- **Navigation Links**:

- **User Actions**:

## Design Specifications

### Color Scheme
- **Primary Color**: `#ff6b35` (Orange)
- **Secondary Color**: `#004e89` (Blue)
- **Accent Color**: `#f77f00` (Amber)
- **Text Dark**: `#1a1a1a`
- **Text Light**: `#666`
- **Background Light**: `#f8f9fa`
- **White**: `#ffffff`

### Typography
- **Font Family**: Inter, Poppins, system fonts
- **Headings**: 700 weight
- **Body**: 400-500 weight
- **Buttons**: 600 weight

### Spacing
- **Section Padding**: 80px vertical
- **Container Max Width**: 1200px
- **Grid Gaps**: 30px
- **Card Padding**: 20px

### Shadows
- **Small**: `0 2px 4px rgba(0,0,0,0.08)`
- **Medium**: `0 4px 12px rgba(0,0,0,0.1)`
- **Large**: `0 8px 24px rgba(0,0,0,0.12)`

### Transitions
- **Duration**: 0.3s
- **Easing**: ease

## Customization Guide

### Changing Colors
Edit CSS variables in `style.css`:
```css
:root {
  --primary-color: #ff6b35;
  --secondary-color: #004e89;
  --accent-color: #f77f00;
}
```

### Adding Products
Add product cards in `index.ejs` following the existing structure:
```html
<div class="product-card">
  <!-- Product content -->
</div>
```

### Modifying Slider
Edit slider items in `index.ejs` and adjust timing in `script.js`:
```javascript
startAutoSlide() {
  autoSlideInterval = setInterval(() => {
    nextSlide();
  }, 5000); // Change interval here
}
```
