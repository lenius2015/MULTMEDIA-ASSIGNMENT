# Admin Panel Design System

## Overview
Modern SaaS-style admin dashboard for E-commerce platform, inspired by Alibaba Admin, Shopify Admin.

## Color Palette
- **Primary Navy**: #1e293b (Dark Blue/Navy)
- **Accent Blue**: #3b82f6 (Blue)
- **Light Gray**: #f1f5f9
- **White**: #ffffff
- **Text Dark**: #0f172a
- **Text Light**: #64748b
- **Success**: #10b981
- **Warning**: #f59e0b
- **Danger**: #ef4444

## Layout
- Sidebar: 280px width, collapsible
- Header: 64px height, fixed top
- Main content: Flexible, with padding

## Components
- Cards: White background, soft shadows, border-radius 8px
- Buttons: Primary (accent blue), Secondary (outline)
- Tables: Striped rows, hover effects
- Forms: Clean inputs with focus states

## Typography
- Font: Inter, sans-serif
- Sizes: Consistent with main site but adapted

## Responsive
- Desktop: Full sidebar
- Tablet: Collapsible sidebar
- Mobile: Hidden sidebar, hamburger menu

## Features
- Dark/Light mode toggle (optional)
- Smooth animations
- Real-time updates
- Toast notifications
- Confirmation modals
